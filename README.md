# eMovieShop

## Project Overview

This project was developed in the context of the **Secure Software Development (DESOFT)** course unit, which challenges students to apply the principles of the **Secure Software Development Life Cycle (SSDLC)** throughout the design and implementation of a real-world application.

### System Description

eMovieShop is a backend REST API built with Java that simulates a movie store. The system supports three user roles (**Customer**, **Support** and
**Admin**) each with a defined set of permissions enforced through Role-Based Access Control (RBAC). 
Authentication is delegated to the external service **Auth0**, with users authenticating via the Authorization Code + PKCE flow and interacting with the backend exclusively through signed JWTs.
